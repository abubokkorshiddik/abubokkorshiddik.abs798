R Package Open Source
whatdis() — R Distribution Detector
Automatically identify and visualize data distributions for statistical modeling decisions
R Statistics EDA

A custom R package designed to help researchers and analysts identify the best-fit statistical distribution for their data. whatdis() automates distribution testing, fitting, and visualization — removing guesswork from exploratory data analysis and improving downstream modeling decisions.

Automatic distribution testing across common families
Visual comparison of fitted distributions
Goodness-of-fit statistics (AIC, BIC, KS test)
Supports skewed, heavy-tailed, and count data
Designed for epidemiological and health data
Minimal dependencies, easy to install

Dengue Analytics Dashboard — Bangladesh
Spatial and temporal visualization of dengue trends and early warning insights
HTML/JS Spatial Analysis Public Health R

An interactive web-based dashboard for analyzing dengue fever trends across Bangladesh using spatial and temporal visualization. Supports district-level early warning insights, climate-disease correlations, and outbreak pattern detection — built to complement published dengue research.

District-level dengue case mapping
Time series trend visualization
Climate variable overlay (temperature, humidity)
Early warning threshold indicators
Responsive design for field use
Linked to published XAI research

HTML/JS PRISMA Literature Review

A web-based research tool designed for efficient systematic review management. Supports literature screening, study filtering, data extraction, and PRISMA-based flowchart generation — enabling transparent and reproducible evidence synthesis for health research.

Title and abstract screening interface
Inclusion / exclusion criteria filtering
PRISMA flowchart generation
Study characteristic data extraction
Export results for reporting
Browser-based, no installation needed
