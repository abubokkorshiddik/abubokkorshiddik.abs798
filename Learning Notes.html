Foundation

What is Statistics?

📖
What it is

The Science of Data

Statistics is the science of collecting, organizing, analyzing, interpreting, and presenting data to make informed decisions and draw meaningful conclusions from uncertainty.

💡
Main Idea

Two Branches

  • Descriptive: Summarizes & describes data (means, charts)
  • Inferential: Draws conclusions about a population from a sample
✅
Where to Use

Applications

  • Medical research & clinical trials
  • Economics & finance forecasting
  • Government census & policy
  • Machine learning & AI
⚠️
Where NOT to Use

Cautions

  • Individual predictions with certainty
  • When data quality is very poor
  • Proving causation from correlation alone
Key Quote "Statistics is the grammar of science." — Karl Pearson. It converts raw numbers into knowledge.

Descriptive Statistics

Measures of Central Tendency

📖
What it is

The "Centre" of Data

A single value that represents the centre or typical value in a dataset. Three primary measures: Mean, Median, Mode. Each is best under different conditions.

🔢
Key Formulas

The Big Three

  • Mean (x̄): Σx / n  (sum ÷ count)
  • Median: Middle value (sorted data)
  • Mode: Most frequent value
  • GM: (x₁·x₂·…·xₙ)^(1/n)
  • HM: n / Σ(1/xᵢ)
✅
When to Use Each

Right Tool, Right Job

  • Mean: Symmetric data, no extreme outliers
  • Median: Skewed data, income/housing prices
  • Mode: Categorical data, most popular item
  • GM: Ratios, growth rates, indices
⚠️
Cautions

Don't Overuse the Mean

  • Mean is highly affected by outliers
  • Mode may not exist or may be multiple
  • Never use mean on nominal/ordinal data
Arithmetic Meanx̄ = (1/n) · Σᵢ xᵢ
Geometric MeanGM = (x₁ · x₂ · … · xₙ)^(1/n)
Harmonic MeanHM = n / (1/x₁ + 1/x₂ + … + 1/xₙ)
RelationHM ≤ GM ≤ AM (always)

Spread of Data

Measures of Dispersion

📏
What it is

How Spread Out Is Data?

Dispersion quantifies the variability or spread in a dataset. Two datasets can have the same mean but very different spreads — dispersion captures this difference.

⚙️
Measures

Types of Dispersion

  • Range: Max − Min (simplest)
  • Quartile Deviation (QD): (Q3−Q1)/2
  • Mean Deviation (MD): Σ|x−x̄| / n
  • Variance (σ²): Σ(x−x̄)² / n
  • Std Deviation (σ): √Variance
  • CV: (σ/x̄)×100 — compares spread
💡
Main Idea

Absolute vs Relative

  • Absolute measures: Range, SD, Variance — in original units
  • Relative measures: Coefficient of Variation (CV) — unit-free, used to compare datasets
Variance (σ²)Σ(xᵢ − x̄)² / n
Std Dev (σ)√[ Σ(xᵢ − x̄)² / n ]
Coeff. of VariationCV = (σ / x̄) × 100%

Bivariate Analysis

Correlation

🔗
What it is

Measuring Association

Correlation measures the strength and direction of the linear relationship between two variables. Value ranges from −1 to +1.

💡
Main Idea

Types of Correlation

  • Positive: Both variables increase together
  • Negative: One increases, other decreases
  • Zero: No linear relationship
  • Perfect: r = ±1 exactly
  • Partial / Multiple: More than 2 variables
⚙️
Methods

How to Compute

  • Pearson's r: For interval/ratio data
  • Spearman's ρ: For ordinal/ranked data
  • Scatter diagram: Visual inspection
  • Karl Pearson's formula most widely used
⚠️
Critical Warning

Correlation ≠ Causation

High correlation does not prove one variable causes the other. A lurking third variable may explain both. Always investigate mechanism, not just numbers.

Pearson's rr = Σ(xᵢ−x̄)(yᵢ−ȳ) / √[Σ(xᵢ−x̄)²·Σ(yᵢ−ȳ)²]
Spearman's ρρ = 1 − (6Σdᵢ²) / (n(n²−1))

Regression Analysis

📉
What it is

Line of Best Fit

Regression establishes a mathematical relationship between variables to predict the value of one variable (dependent, Y) from another (independent, X).

⚙️
How it Works

Ordinary Least Squares

We minimize the sum of squared residuals (errors) between observed Y and predicted Ŷ. This gives us the unique best-fit line.

  • Two regression lines: Y on X, X on Y
  • They intersect at (x̄, ȳ)
💡
Relationship with r

Coefficient Link

  • r² = b_yx × b_xy (product of regression coefficients)
  • r = √(b_yx × b_xy)
  • Both byx and bxy have the same sign as r
Regression LineŶ = a + bX
Slope (b)b = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)²
Intercept (a)a = ȳ − b·x̄

Methods

Association

Two attributes are associated if they are not independent — their joint frequency differs from what would be expected by chance. Positive / negative / zero association.

Foundations

Set Theory

📖
What it is

Sets & Notation

  • Set: A well-defined collection of distinct objects (elements)
  • Notation: Sets in capital letters {A, B, C}; elements in lowercase {a, b, c}
  • Roster form: A = {1, 2, 3}
  • Set-builder form: A = {x : x is a positive integer < 4}
💡
Types of Sets

Key Set Vocabulary

  • Universal set (U / Ω): Contains all elements under discussion
  • Empty set (∅ / {}): No elements; unique
  • Finite set: Countable number of elements
  • Infinite set: Uncountable (e.g., all real numbers)
  • Unit set (Singleton): Exactly one element
⚙️
Subset Relations

Subsets & Equality

  • Subset (⊆): A ⊆ B means every element of A is in B
  • Proper subset (⊂): A ⊂ B and A ≠ B
  • Equal sets: A = B iff A ⊆ B and B ⊆ A
  • ∅ is a subset of every set
  • Every set is a subset of itself
🔢
Power Set & Cartesian

Derived Sets

  • Power set P(A): Set of all subsets of A; |P(A)| = 2ⁿ
  • Cartesian product: A × B = {(a,b) : a∈A, b∈B}
  • Class of sets: A set whose elements are themselves sets
✅
Set Operations

Union, Intersection, Difference

  • Union (A ∪ B): Elements in A or B or both
  • Intersection (A ∩ B): Elements in both A and B
  • Complement (Aᶜ): Elements in U but not in A
  • Difference (A − B): In A but not in B
  • Symmetric diff. (A △ B): (A−B) ∪ (B−A)
🔵
Venn Diagrams

Visual Representation

Venn diagrams use overlapping circles inside a rectangle (universal set) to visualize set relationships. Regions represent: A only, B only, A∩B, and neither. Extremely powerful for verifying laws and solving counting problems.

UnionA ∪ B = {x : x ∈ A or x ∈ B}
IntersectionA ∩ B = {x : x ∈ A and x ∈ B}
ComplementAᶜ = {x ∈ U : x ∉ A}
DifferenceA − B = A ∩ Bᶜ
Power Set Size|P(A)| = 2^|A|

Laws of Sets

Algebra of Sets & De Morgan's Laws

📐
Commutative Laws

Order Doesn't Matter

  • A ∪ B = B ∪ A
  • A ∩ B = B ∩ A
🔗
Associative Laws

Grouping Doesn't Matter

  • (A ∪ B) ∪ C = A ∪ (B ∪ C)
  • (A ∩ B) ∩ C = A ∩ (B ∩ C)
📦
Distributive Laws

Distributing Over Operations

  • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
  • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
🏛️
Identity & Idempotent

Self & Identity Laws

  • Identity: A ∪ ∅ = A  |  A ∩ U = A
  • Idempotent: A ∪ A = A  |  A ∩ A = A
  • Complement: A ∪ Aᶜ = U  |  A ∩ Aᶜ = ∅
  • Double complement: (Aᶜ)ᶜ = A
De Morgan's Laws — The Most Important Laws The complement of a union is the intersection of complements: (A ∪ B)ᶜ = Aᶜ ∩ Bᶜ. The complement of an intersection is the union of complements: (A ∩ B)ᶜ = Aᶜ ∪ Bᶜ. These are used constantly in probability to find complements of complex events.
De Morgan I(A ∪ B)ᶜ = Aᶜ ∩ Bᶜ
De Morgan II(A ∩ B)ᶜ = Aᶜ ∪ Bᶜ
Inclusion-Exclusion|A ∪ B| = |A| + |B| − |A ∩ B|

Core Theory

Probability Fundamentals

📖
What it is

Probability vs Possibility

  • Possibility: Something can happen (yes/no)
  • Probability: How likely it is (a number 0 to 1)
  • P = 0 → impossible; P = 1 → certain
  • Probability quantifies uncertainty numerically
🎲
Experiment & Sample Space

Building Blocks

  • Random experiment: Outcome cannot be predicted with certainty
  • Sample space (S / Ω): Set of all possible outcomes
  • Event (A): A subset of the sample space
  • Elementary event: A single outcome {ω}
  • Complementary event: Aᶜ — all outcomes not in A
💡
Basic Requirements

3 Axioms of Probability

  • Axiom 1: P(A) ≥ 0 for any event A
  • Axiom 2: P(S) = 1 (the whole sample space)
  • Axiom 3: For mutually exclusive events: P(A∪B) = P(A) + P(B)
🔢
Mutually Exclusive

Special Event Types

  • Mutually exclusive: A ∩ B = ∅ (can't both happen)
  • Exhaustive: A ∪ B = S (cover all outcomes)
  • Equally likely: All outcomes have same probability
Probability of AP(A) = Number of favourable outcomes / Total outcomes
Complement RuleP(Aᶜ) = 1 − P(A)
Bounds0 ≤ P(A) ≤ 1    always

P4
Approaches

Methods of Assigning Probability

🏛️
Classical Method

Equally Likely Outcomes

P(A) = m/n where m = favourable outcomes, n = total equally likely outcomes. Assumes outcomes are equally likely. Best for dice, coins, cards.

⚠ Fails when outcomes are not equally likely

📊
Empirical / Relative Frequency

Based on Observation

P(A) = lim (f/n) as n→∞, where f = frequency of A in n trials. Derived from actual data. Used in insurance, quality control.

⚠ Requires large number of trials

📐
Geometric Method

Area / Length Ratio

Used when the sample space is continuous. P(A) = measure of A / measure of S. Example: randomly picking a point inside a circle.

🔢
Axiomatic Method

Kolmogorov's Axioms

Defines probability via 3 axioms (non-negativity, normalisation, countable additivity). The most rigorous and general foundation — underpins all modern probability theory.

🧠
Subjective Method

Personal Degree of Belief

Probability assigned based on expert judgment, opinion, or past experience when data is unavailable. Common in Bayesian statistics and forecasting.

⚠ Not objective — varies between individuals

💡
Probability & Odds

Odds Ratio

  • Odds in favour of A: P(A) / P(Aᶜ)
  • Odds against A: P(Aᶜ) / P(A)
  • If odds in favour = a : b → P(A) = a/(a+b)

P5
Combinatorics

Addition Law & Counting Rules

➕
Addition Law

Union of Events

  • General: P(A∪B) = P(A) + P(B) − P(A∩B)
  • Mutually exclusive: P(A∪B) = P(A) + P(B)
  • Three events: P(A∪B∪C) = P(A)+P(B)+P(C) −P(A∩B)−P(B∩C)−P(A∩C)+P(A∩B∩C)
🔢
Counting Rules

Fundamental Principle

  • Multiplication rule: m × n ways to do two independent tasks
  • Permutation P(n,r): Ordered arrangements of r from n
  • Combination C(n,r): Unordered selections of r from n
  • Tree diagram: Visual enumeration of all outcomes
🌳
Tree Diagrams

Mapping All Paths

A tree diagram systematically lists all possible outcomes of sequential events. Each branch represents one choice; multiply branch probabilities along any path for that path's probability. Invaluable for compound events.

Addition (general)P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
PermutationP(n,r) = n! / (n−r)!
CombinationC(n,r) = n! / [r! · (n−r)!]

P6
Updated Belief

Conditional Probability & Independence

🔍
What it is

Probability Given Info

P(A|B) is the probability of A given that B has already occurred. We restrict the sample space to B and measure A within it. This is "updated" probability with new information.

⚙️
How it Works

Multiplication Rule

  • P(A|B) = P(A ∩ B) / P(B), provided P(B) > 0
  • P(A ∩ B) = P(B) · P(A|B) = P(A) · P(B|A)
  • This is the general multiplication rule
💡
Independence

When Knowledge Changes Nothing

  • A and B are independent if P(A|B) = P(A)
  • Equivalently: P(A ∩ B) = P(A) · P(B)
  • Non-independent (dependent): Knowing B changes P(A)
  • Independence ≠ Mutual exclusivity
⚠️
Common Confusion

Independence vs Exclusivity

Mutually exclusive events (A ∩ B = ∅) with positive probability are NEVER independent — if A occurs, B definitely cannot. Independence means learning A gives no information about B.

Conditional Prob.P(A|B) = P(A ∩ B) / P(B)
Multiplication RuleP(A ∩ B) = P(A) · P(B|A)
Independence TestP(A ∩ B) = P(A) · P(B)

P7
Inverse Probability

Bayes' Theorem

🔄
What it is

Reversing Conditional Probability

Bayes' theorem allows us to find P(cause | effect) when we know P(effect | cause). We update our prior belief about a hypothesis after observing new evidence.

⚙️
How it Works

Prior → Posterior

  • Prior P(Hᵢ): Initial belief before evidence
  • Likelihood P(E|Hᵢ): Probability of evidence given hypothesis
  • Posterior P(Hᵢ|E): Updated belief after seeing evidence
  • Denominator = total probability of evidence
✅
Applications

Where Bayes is Used

  • Medical diagnosis (P(disease | positive test))
  • Spam email filtering
  • Machine learning classifiers
  • Legal reasoning & forensics
  • Weather forecasting updates
Bayes' TheoremP(Hᵢ|E) = P(E|Hᵢ)·P(Hᵢ) / Σⱼ P(E|Hⱼ)·P(Hⱼ)
Total ProbabilityP(E) = Σⱼ P(E|Hⱼ)·P(Hⱼ)
Simple FormP(H|E) = P(E|H)·P(H) / P(E)
Intuition Bayes' theorem is the formal way to update your beliefs with new evidence. If a medical test is positive, Bayes tells you the true probability of actually having the disease, accounting for the test's false positive rate and disease prevalence.

Extended Probability

Total & Compound Probability

🌐
Total Probability

Law of Total Probability

If {H₁, H₂, …, Hₙ} is a partition of S (mutually exclusive & exhaustive), then for any event E: P(E) = Σ P(E|Hᵢ)·P(Hᵢ). We "average" over all possible causes.

⚙️
Compound Probability

Sequential Events

  • P(A ∩ B ∩ C) = P(A)·P(B|A)·P(C|A∩B)
  • For independent events: P(A ∩ B ∩ C) = P(A)·P(B)·P(C)
  • Used for probability along tree diagram paths
🎯
Odds Ratio Revisited

Expressing Probability as Odds

  • Odds in favour = P(A) : P(Aᶜ) = P(A) : (1−P(A))
  • Posterior odds = Prior odds × Likelihood ratio (Bayes factor)
  • Used in gambling, medicine, epidemiology
Total ProbabilityP(E) = Σᵢ P(E|Hᵢ) · P(Hᵢ)
Chain RuleP(A∩B∩C) = P(A) · P(B|A) · P(C|A∩B)
Odds in FavourOdds(A) = P(A) / (1 − P(A))

Core Concept

Random Variables

🎯
What it is

Mapping Outcomes to Numbers

A random variable (RV) X is a function X: S → ℝ that assigns a real number to each outcome in the sample space. It translates non-numeric experiments into numerical values for mathematical analysis.

💡
Two Types

Discrete vs Continuous

  • Discrete RV: Takes countable values (0, 1, 2, …) — e.g., number of heads, defective items
  • Continuous RV: Takes any value in an interval — e.g., height, time, temperature
  • Distinction determines which tools to use
📈
Distribution Function

CDF — Cumulative Distribution

  • CDF F(x): P(X ≤ x) for any x
  • Always non-decreasing; F(−∞) = 0; F(+∞) = 1
  • Right-continuous for discrete; continuous for continuous RV
🔢
PMF & PDF

Probability Functions

  • PMF p(x): P(X = x) for discrete — must sum to 1
  • PDF f(x): For continuous — area under curve = 1
  • P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx for continuous
🔄
Function of a RV

Transformations

If Y = g(X), then Y is also a random variable. We find its distribution by transformation methods. Example: if X ~ N(0,1), then X² ~ χ²(1). Transformations are fundamental in deriving new distributions.

⚠️
Where NOT to Confuse

RV vs Its Values

X (capital) is the random variable — the function. x (lowercase) is the specific value it takes. Writing P(X = x) properly distinguishes between the function and its realization.

CDFF(x) = P(X ≤ x)
PMF (discrete)Σₓ p(x) = 1   and   p(x) ≥ 0
PDF (continuous)∫₋∞^∞ f(x) dx = 1   and   f(x) ≥ 0
CDF from PDFF(x) = ∫₋∞^x f(t) dt

📊
What it is

Complete Probability Model

A probability distribution fully describes how probabilities are spread across all possible values of a random variable. It is the complete "profile" of uncertainty for that variable.

💡
Key Properties

Valid Distribution Conditions

  • All probabilities ≥ 0
  • All probabilities sum (or integrate) to 1
  • Uniquely identifies the random variable's behaviour
  • Allows computing mean, variance, percentiles
⚙️
Common Characteristics

Summarizing a Distribution

  • Mean (μ): Centre / expected value
  • Variance (σ²): Spread around mean
  • Skewness: Asymmetry of the distribution
  • Kurtosis: Tail heaviness
  • MGF / CF: Moment generating function

Multivariate

Joint, Marginal & Conditional Distributions

🔗
Joint Distribution

Two Variables Together

The joint distribution of (X, Y) gives P(X=x, Y=y) for discrete, or f(x,y) for continuous. It captures the full relationship between two random variables simultaneously.

📉
Marginal Distribution

Summing Out One Variable

  • Discrete: p_X(x) = Σᵧ p(x, y)
  • Continuous: f_X(x) = ∫ f(x,y) dy
  • Gives the distribution of each variable alone, ignoring the other
💡
Conditional Distribution

Given the Other Variable

  • f(y|x) = f(x,y) / f_X(x)
  • Describes Y's behaviour when X is fixed at x
  • If independent: f(x,y) = f_X(x) · f_Y(y)
⚖️
Independence of RVs

When They Don't Interact

  • X and Y are independent iff f(x,y) = f_X(x)·f_Y(y)
  • For independent: E(XY) = E(X)·E(Y)
  • Cov(X,Y) = 0 if independent (but not always vice versa)
Joint PMFp(x,y) = P(X=x, Y=y)
Marginal (discrete)p_X(x) = Σᵧ p(x,y)
Conditionalf(y|x) = f(x,y) / f_X(x)
Independencef(x,y) = f_X(x) · f_Y(y)

P12
Moments & Summary

Mathematical Expectation

⚖️
What it is

Weighted Average of Values

The expected value E(X) is the long-run average of X over infinitely many repetitions. It is the probability-weighted average of all possible values — the "centre of gravity" of a distribution.

⚙️
Key Properties

Linearity of Expectation

  • E(aX + b) = aE(X) + b
  • E(X + Y) = E(X) + E(Y)  (always)
  • E(XY) = E(X)·E(Y)  only if independent
  • Var(X) = E(X²) − [E(X)]²
💡
Moments

rth Order Moments

  • rth raw moment: μ'ᵣ = E(Xʳ)
  • rth central moment: μᵣ = E[(X − μ)ʳ]
  • 1st: μ = E(X) (mean)
  • 2nd central: σ² = Var(X)
  • 3rd: Skewness; 4th: Kurtosis
🔢
Covariance & Correlation

Between Two RVs

  • Cov(X,Y) = E(XY) − E(X)·E(Y)
  • ρ(X,Y) = Cov(X,Y) / (σ_X · σ_Y)
  • Var(aX + bY) = a²Var(X) + b²Var(Y) + 2ab·Cov(X,Y)
E(X) — discreteE(X) = Σₓ x · p(x)
E(X) — continuousE(X) = ∫₋∞^∞ x · f(x) dx
VarianceVar(X) = E(X²) − [E(X)]²
CovarianceCov(X,Y) = E[(X−μₓ)(Y−μᵧ)]

Named Distributions

Basic Probability Distributions

Bernoulli Binomial Poisson Geometric Negative Binomial Hypergeometric Uniform Normal Exponential Gamma Beta
🪙
Discrete — Bernoulli & Binomial

Success/Failure Trials

  • Bernoulli(p): Single trial; X ∈ {0,1}; E=p, Var=p(1−p)
  • Binomial(n,p): n independent Bernoulli trials; counts successes
  • P(X=k) = C(n,k)·pᵏ·(1−p)ⁿ⁻ᵏ
  • E(X)=np, Var(X)=np(1−p)
☎️
Discrete — Poisson

Rare Events in Time/Space

  • Models count of events in a fixed interval
  • P(X=k) = e⁻λ · λᵏ / k!
  • E(X) = Var(X) = λ (equal — unique property!)
  • Used for: calls/hour, defects/unit, accidents/year
🔔
Continuous — Normal

The Bell Curve

  • X ~ N(μ, σ²); symmetric about μ
  • 68–95–99.7 rule for 1, 2, 3 standard deviations
  • Standard normal: Z ~ N(0,1)
  • Central Limit Theorem: sums → Normal
⏱️
Continuous — Exponential

Time Until First Event

  • f(x) = λe⁻λˣ for x ≥ 0
  • E(X) = 1/λ; Var(X) = 1/λ²
  • Memoryless property: P(X>s+t | X>s) = P(X>t)
  • Used for: equipment failure times, wait times
🎰
Geometric & Neg. Binomial

Waiting for Success

  • Geometric: Trials until first success; P(X=k)=(1−p)ᵏ⁻¹·p
  • Neg. Binomial: Trials until rth success
  • Both have the memoryless property (geometric)
🧮
Hypergeometric

Sampling Without Replacement

  • N items total; K successes in population
  • Draw n items; X = successes drawn
  • No replacement → not Binomial
  • Used in quality control, card games
Binomial PMFP(X=k) = C(n,k) · pᵏ · (1−p)ⁿ⁻ᵏ
Poisson PMFP(X=k) = e⁻λ · λᵏ / k!
Normal PDFf(x) = (1/σ√2π) · exp[−(x−μ)²/2σ²]
Exponential PDFf(x) = λ · e⁻λˣ , x ≥ 0
Which distribution to choose? Binary outcome single trial → Bernoulli. Counting successes in n trials (with replacement) → Binomial. Counting rare events in time/space → Poisson. Waiting time between events → Exponential. Counting without replacement → Hypergeometric. Heights, measurement errors, averages → Normal.
